Button(props, …props) → {JSX.Element}

Botón reutilizable con variantes de estilo, tamaño y opción de bloque completo.
Parameters:
Name Type Attributes Description
props object Props del componente, es el objeto en el que se definen todas las propiedades del botón.
Properties
Name Type Attributes Default Description
children React.ReactNode Cualquier contenido que se introduzca entre las etiquetas del botón.
variant 'primary' | 'secondary' | 'link' | 'ghost' <optional>
 'primary' Son los estilos predefinidos, en este caso de bootstrap. Internamente se traduce en una clase de Bootstrap que asigna colores de fondo, borde y comportamiento al pasar el ratón.
size 'sm' | 'md' | 'lg' <optional>
 'md' Controlamos el padding y el tamaño de fuente para adaptarlo a un botón pequeño(sm), mediano(md) o grande(lg). Por defecto es mediano(md).
block boolean <optional>
 false Si es true se añadiría la clase w-full para que el botón ocupe todo el ancho de su contenedor.
className string <optional>
 Esto basicamente nos permite añadir clases CSS adicionales en caso de que necesitásemos personalizar más allá de las variantes y tamaños predeterminados.
props any <repeatable>
 Captura cualquier otra propiedad que pudieramos necesitar pasar al elemento

El resultado que devolveremos es un botón estilizado con las clases de Bootstrap y los estilos aplicados.
